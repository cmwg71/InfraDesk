# Aufgabenstellung: Barrierefreiheit (Accessibility)

### Beschreibung

Optimierung der WinUI-Benutzeroberfläche für eine inklusive Nutzung und Einhaltung von Standards.

### Funktionsumfang

1. **Tastatur-Navigation**:
    
    - Durchgängige Tab-Indizes und Fokus-Visualisierung in allen Dialogen.
        
2. **Screenreader-Unterstützung**:
    
    - Korrekte Verwendung von `AutomationProperties.Name` und Rollen für UI-Elemente.
        
3. **Kontraste & Themes**:
    
    - Unterstützung von Windows Dark/Light Mode.
        
    - Einhaltung der WCAG 2.1 AA Kontrastverhältnisse.
        
4. **Skalierbarkeit**:
    
    - UI-Layouts müssen bei Textskalierung (bis 200%) benutzbar bleiben.
        

### Abnahmekriterien

- Die App lässt sich vollständig ohne Maus bedienen.
    
- Der "Accessibility Insights" Test besteht die Standard-Prüfungen.

# Aufgabenstellung: Knowledge Base (KB)

### Beschreibung

Aufbau einer Wissensdatenbank zur Reduzierung von wiederkehrenden Tickets und zur Dokumentation von Lösungen.

### Funktionsumfang

1. **Artikel-Editor**:
    
    - WYSIWYG-Editor für Artikel mit Unterstützung für Bilder und eingebettete Videos.
        
2. **Verknüpfung**:
    
    - Artikel können mit Asset-Kategorien (z. B. "Handbuch für Drucker X") oder Problemen (T2_05) verlinkt werden.
        
3. **Suche**:
    
    - Integration der KB-Artikel in die globale Suche (T1_11).
        
4. **Feedback-System**:
    
    - Nutzer können Artikel als "hilfreich" markieren oder kommentieren.
        

### Abnahmekriterien

- In einem Ticket lassen sich passende KB-Artikel direkt als Lösung vorschlagen.
    
- Die Suche liefert relevante KB-Einträge priorisiert zurück.

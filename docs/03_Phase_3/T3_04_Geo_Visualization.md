# Aufgabenstellung: Geografische Visualisierung

### Beschreibung

Räumliche Darstellung der IT-Infrastruktur auf Karten und innerhalb von Gebäuden (Indoor-Mapping).

### Funktionsumfang

1. **Weltkarte (GIS)**:
    
    - Integration von OpenStreetMap/Leaflet in WinUI und Web-Portal.
        
    - Anzeige von Standorten als Marker mit Status-Indikatoren (z. B. Rot bei kritischen Tickets).
        
2. **Etagenpläne (Floorplans)**:
    
    - Upload von SVG- oder Bilddateien für Gebäudepläne.
        
    - Platzierung von Assets (Drucker, Switche, APs) per Drag & Drop auf dem Plan.
        
3. **Live-Tracking (optional)**:
    
    - Anzeige der letzten bekannten Position von mobilen Geräten (via Discovery-Daten).
        

### Abnahmekriterien

- In der Standort-Übersicht wird eine interaktive Karte angezeigt.
    
- Assets lassen sich in einer "Grundriss-Ansicht" exakt verorten.
